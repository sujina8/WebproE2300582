EduSkill System
An online course system connecting learners with certified training providers in Malaysia.
Built for BIT210 Web Programming at HELP University.

Tech Stack
HTML5 CSS3 Bootstrap 5 JavaScript PHP 8 MySQL XAMPP

Features
Students can browse, enrol, and review courses
Training Providers can list and manage courses
Ministry Admin can approve providers and view reports
